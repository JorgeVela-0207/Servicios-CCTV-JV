# Servicios CCTV JV

Landing page para **Servicios CCTV JV**: instalación de cámaras de seguridad y videovigilancia para casa y pequeños negocios.

Sitio estático, responsive y orientado a cotizaciones por WhatsApp.

## Servicios

- Instalación de sistemas CCTV
- Configuración de acceso remoto desde móvil
- Mantenimiento y soporte técnico
- Suministro e instalación de equipos

## Marcas

Trabajo con equipos originales de **Hikvision**, **Dahua** e **HiLook**, seleccionados según cada proyecto.

## Opciones de instalación

| Paquete | Referencia |
|---------|------------|
| **Básica** | Desde 2 cámaras, grabación local y acceso móvil |
| **Intermedia** | Desde 4 cámaras, grabación extendida y alertas |
| **Avanzada** | Desde 8 cámaras, soluciones personalizadas |

> Las cotizaciones se realizan de forma personalizada y sin compromiso.

## Ver el sitio

**En local:** abre `index.html` en tu navegador.

**Con servidor local (opcional):**

```bash
python -m http.server 8080
```

Luego visita `http://localhost:8080`.

## Tecnologías

- HTML5
- CSS3
- JavaScript
- [Font Awesome](https://fontawesome.com/) (iconos vía CDN)

No requiere Node.js, npm ni proceso de build.
